Transportation System is a Django-based web application designed to manage and optimize transportation operations. Key features and highlights include:

Real-time Bus Tracking: View accurate arrival and departure times for buses.

Route Management: Add, edit, or remove bus routes efficiently.

User Authentication: Secure login and registration for administrators and passengers.

QR Code Integration: Generate and scan QR codes for quick access to bus and route information.

Student/Passenger Registration: Maintain organized records of registered users per bus.

Feedback Collection: Users can submit feedback easily through the system.

Database Management: Uses SQLite for reliable storage of all system data.

Responsive Frontend: Built with HTML, CSS, and JavaScript for a clean and user-friendly interface.

Administrative Control: Track buses, routes, and user activity seamlessly.

Easy Setup: Install dependencies via requirements.txt and run locally with Django’s development server.
